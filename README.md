# 利用方法

### できるかな
１．年齢帯・カテゴリ・難易度を切り替える。
２．各カードのタスクを読み、「がんばり中」「できた！」のボタンを押す。３．付いた花丸アイコンを増やしていき、花丸が一定数に到達するとメダルやトロフィーが解放される。

### そうだん
１．育児の悩みやタスクの解決方法について入力する。
２．返ってきたAIの答えを確認する。

### レポート
１．頑張った功績をA4で印刷したり、PDF保存したりする。


# アプリケーションを作成した背景

「何歳で何ができると良いか」を分かりやすく示すことで、保護者の方々の不安を減らし、子どもたちが適切な成長の機会を得られるようにしたいという願いから着想した。SNSのような公開性や点数化ではなく、家庭の記録にフォーカスし、子どもの成長を記録していってほしいと思っている。

# 工夫したポイント

・アプリの良さを知ってもらうことが先決と考え、ログインしなくてもある程度の機能が使えるようにした。
・利用者のストレスにならないよう、「がんばり中」「できた！」ボタンを押してもページが遷移しないようにした。
・利用者同士の意見のやり取りによる衝突やストレスを回避するために、コメントやチャット機能を取り入れるのではなく、AIによる相談チャットという形にした。
・利用者が安心できるよう、やわらかい配色（サニー×はちみつ色）にした。


# 改善点

・できるかな機能のタスクが１００件あるが、並べられたカードをズラッと見ると単調的に感じてしまうこと。イラストを添えたり、カテゴリ別に色の違いを出したりしないと、どれも同じタスクに見えてしまう。
